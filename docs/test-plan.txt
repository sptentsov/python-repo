1.Главная страница, на ней должно быть
	1.1. Поле для ввода логина вверху экрана
	1.2. Поле для ввода пароля вверху экрана
	1.3. Кнопка входа вверху экрана. Реакция на нажатие - см. пункт 3
	1.4 Кнопка для регистрации вверху экрана. Реакция на нажатие - см. пункт 2.
	1.5. Описание сервиса
	1.6. Большая кнопка регистрации рядом с описанием сервиса. Реакция на нажатие - см. пункт 2.
2.Регистрация
	2.1. При вызове регистрации открывается страница регистрации. (п. 6.)
3.Вход
	3.1. Установить куки
4.Выход
	4.1. Удалить куки
	4.2. Перенаправить на главную страницу (п.1.)
5.Страница мувиков
	5.1. Лист мувиков
		5.1.1. Изначально лист пуст
			*5.1.1.1. В нем находится кнопка у вас нет мувиков, хотите добавить?
		5.1.2. Список названий мувиков
			5.1.2.1. При щелчке на название мувика он открывается в окне мувика		
	5.2. Кнопка добавления мувика
		5.2.1. При щелчке на неё открывается окно мувика со всеми пустыми полями
	5.3. Окно мувика
		5.3.1. Название мувика
			5.3.1. При нажатии на него оно становится полем для ввода, появляется кнопка ок
			5.3.2. При нажатии на ок название возвращается в исходное состояние, его текст сохраняется, изменяется.
		5.3.2. Блок (блоки) сезонов
			5.3.2.1. Нижний блок всегда блок добавления сезона.
				5.3.2.1.1. На нем кнопка, которая по нажатию добавляет пустой блок над блоком добавления сезона.
			5.3.2.2. На остальных блоках отображается содержимое блока
				5.3.2.2.1. Добавленные серии
					При щелчке на добавленную серию изменяется отметка её (смотрел/нет)
				5.3.2.2.2. Кнопка добавления одной серии
					При нажатии на нее должна куда надо добавляться непросмотренная серия
				*5.3.2.2.3. Добавление сразу многих серий
				*5.3.2.2.3. Удаление серий
				5.3.2.2.4. Номер блока сезона
				*5.3.2.2.5. Название сезона
6.Страница регистрации
	6.1. Поле для ввода логина
	6.2. Поле для ввода пароля
	6.3. Поле для ввода подтверждения пароля.
	6.4. Кнопка "Ок", при нажатии:
		6.4.1. Проверка занятости логина, совпадения версий пароля. 
		6.4.2. Если все ок, то выполнить действие вход (п. 3.), перенаправить юзера на страницу мувиков.(п. 5.)
